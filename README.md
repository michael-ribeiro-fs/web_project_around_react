## ✨ Funcionalidades implementadas

- ✅ Estrutura modular com componentes React funcionais
- ✅ Renderização de cartões a partir de dados mock
- ✅ Abertura de popups para:
  - Editar perfil
  - Alterar avatar
  - Adicionar novo cartão
  - Visualizar imagem ampliada
- ✅ Fechamento de popups via botão, tecla `ESC` ou clique no overlay
- ✅ Layout responsivo
- ✅ Gerenciamento de estado com `useState` e efeitos com `useEffect`

## 🚀 Como executar

```bash
# Clone o repositório
git clone https://github.com/michael-ribeiro-fs/web_project_around_react

# Acesse a pasta
cd web_project_around_react

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```
